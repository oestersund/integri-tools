# INTEGRI Tools

A collection of standalone tools for marketing and business development at INTEGRI. This repository hosts web-based applications designed to be served directly from GitHub Pages.

## Contents

- **Website Briefing** (`website-briefing/`): Comprehensive marketing briefing form with auto-save, JSON export/import, PDF generation, and email integration.

## Deployment

Deploy to GitHub Pages or use the custom domain configured in `CNAME` (tools.integri.de).
